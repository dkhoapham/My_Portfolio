# Portfolio Website

This is my personal portfolio, built with **Vite**, **React**, **TypeScript**, and **Tailwind CSS**.  
It’s a place where I showcase my work and continue refining how I approach frontend engineering, UI design, and component architecture.

🔗 Live site: https://dkhoapham.com

---

## Tech Stack

- **React + TypeScript** — component-driven UI
- **Vite** — fast development server and optimized builds
- **Tailwind CSS** — utility-first styling with a custom design system
- **Framer Motion** — transitions and micro‑interactions
- **Vercel** — hosting and deployment

---

## Project Focus

This project isn’t just a portfolio — it’s an ongoing exercise in building clean, maintainable frontend code.  
Some of the things I focus on:

- Consistent spacing, layout, and visual rhythm
- Clear component boundaries and reusable patterns
- Smooth transitions that feel intentional
- A responsive design that works across devices
- A simple, reliable deployment workflow

---

## Features

- Fully responsive layout
- Modular component structure
- Page transitions and motion primitives
- Custom color system and theming
- Dedicated 404 page
- Polished navigation and alignment

---

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
